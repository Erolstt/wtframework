Wtframework

Framework for structured Web Tests in Python.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Wtframework.

The easy way to organize your web tests. WTF aims to be the glue that holds together all the open source pieces needed to test a Web App in Python.

Installation

Requirements

  • Python 2.7 - http://www.python.org/download/
  • PyPi (pip) - http://www.pip-installer.org/en/latest/

Installation via PYPI

pip install wtframework

See installation instructions, https://github.com/wiredrive/wtframework/wiki/Installation-Guide
